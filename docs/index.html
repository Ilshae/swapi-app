Star Wars API Documentation

This documentation provides an overview of the Star Wars API, its queries, and mutations.

API Endpoints
http://localhost:3000/graphql

Queries

film

Description

Retrieve a single film by its ID

Response

Returns a Film!

Arguments
Name Description
id - Float! ID of the film to retrieve

Example

Query
query film($id: Float!) {
  film(id: $id) {
    title
    director
    producer
    release_date
  }
}
Variables
{"id": 987.65}
Response
{
  "data": {
    "film": {
      "title": "xyz789",
      "director": "abc123",
      "producer": "xyz789",
      "release_date": "xyz789"
    }
  }
}
Queries

films

Description

Retrieve all films with optional pagination and filtering

Response

Returns [Film!]!

Arguments
Name Description
page - Float Page number for pagination
filter - String Keyword to filter films

Example

Query
query films(
  $page: Float,
  $filter: String
) {
  films(
    page: $page,
    filter: $filter
  ) {
    title
    director
    producer
    release_date
  }
}
Variables
{"page": 123.45, "filter": "xyz789"}
Response
{
  "data": {
    "films": [
      {
        "title": "xyz789",
        "director": "abc123",
        "producer": "abc123",
        "release_date": "abc123"
      }
    ]
  }
}
Queries

openingCrawlAnalysis

Description

Analyze opening crawls

Response

Returns an OpeningCrawlAnalysis!

Example

Query
query openingCrawlAnalysis {
  openingCrawlAnalysis {
    wordCounts {
      ...WordCountFragment
    }
    mostMentionedCharacters
  }
}
Response
{
  "data": {
    "openingCrawlAnalysis": {
      "wordCounts": [WordCount],
      "mostMentionedCharacters": ["xyz789"]
    }
  }
}
